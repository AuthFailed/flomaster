### Проверка версии МП
```txt
В приложении "Умный Дом.ру" откройте раздел "Кабинет", далее пункт "О приложении"
Уточните, какая версия приложения там указана?
```

### Добавление камеры в МП
```txt
Для добавления камеры:
1. Находясь на главном экране, в блоке "Мой дом", нажмите на иконку "+"
2. Выберите из списка модель вашей камеры;
3. Появится окно выбора тарифа, выбираем указанный по договору тариф и принимаем условия Публичной оферты;
4. Выберите способ подключения камеры к сети, кабелем или Wi-Fi и подключитесь к сети.
```
