#### Смена каналов
```txt
1. Подключите устройство к роутеру кабелем или по сети Wi-Fi;
2. В адресной строке любого браузера введите: 192.168.0.1 и перейдите на страницу;
3. Должно появиться окно настроек роутера, при запросе пароля указываем admin или admin1 (если свой пароль не ставили);

Переходим в раздел "Wi-FI", затем выбираем вашу частоту сети
В открывшемся окне нам нужна строчка "Номер канала", меняем в ней значение на

Настройки сохраняем
```

#### Настройки Wi-Fi
```txt
1. Подключите устройство к роутеру кабелем или по сети Wi-Fi;
2. В адресной строке любого браузера введите: 192.168.0.1 и перейдите на страницу;
3. Должно появиться окно настроек роутера, при запросе пароля указываем admin или admin1 (если свой пароль не ставили);

Переходим в раздел "Wi-FI", затем выбираем вашу частоту сети
В строчке "SSiD" можно изменить название сети
Для изменения пароля от Wi-Fi откройте подпункт "Безопасность", новый пароль вводим в строчке "Предварительный общий ключ"

Настройки сохраняем
```

#### Настройка моста
Для прошивки EC220
```txt
Для настройки Bridge:
1. Перейдите в раздел "Дополнительные настройки";
2. Выберите в списке слева пункт "Сеть" - "Интернет";
3. Нажмите "Добавить", в появившемся окне настройки типа соединения выберите "Мост" и нажмите "Сохранить";
4. Далее перейдите в раздел "Группировка интерфейсов" и нажмите "Добавить"

В открывшихся настройках выберите порт LAN для подключения VoIP-шлюза и созданное до этого Bridge-соединение
Также нужно задать имя для группы портов (например VoIP). Затем сохраните настройки
```

#### Настройка Wi-Fi
```txt
Откройте слева меню "Беспроводная сеть" в разделе "Дополнительные настройки"
На вкладке установите следующие опции:
- SSID - желаемое название Wi-Fi сети;
- Скрыть SSID - нет;
- Метод проверки подлинности - WPA2-Auto-Personal;
- Предварительный ключ WPA - желаемый пароль Wi-Fi сети.

Далее нажмите "Применить"
```

#### Включение IGMP
```txt
Слева, в разделе "Дополнительные настройки, выберите пункт "Локальная сеть"
Затем во вкладке IPTV установите галочку на пунктах:
- Включить многоадресную маршрутизацию (IGMP Proxy);
- Enable efficient multicast forwarding (IGMP Snooping).

Далее нажмите "Применить"
```

#### Настройка PPPoE Relay
```txt
Слева выберите раздел "Интернет", потом в меню сверху вкладку "NAT Passthrough"
Внизу будет опция "Включить ретрансляцию PPPoE", включите ее

Далее нажмите "Применить"
```

#### Настройка DMZ
```txt
Перейдите слева в раздел "Интернет", далее посередине сверху выберите вкладку "DMZ"
Установите следующие опции на странице:
- "Включить DMZ" - установить в "Да";
- IP-адрес видимой станции - установите в IP конечного устройства.

Далее нажмите "Применить"
```
