### Проверка наличия разрывов по патч-корду
```txt
Сложность с разрывами соединения наблюдается только на устройствах по вайфай?
Или же на устройствах, подключенных кабелем, так же интернет пропадает?
```

### Перенастройка роутера
#### Предложение перенастройки (Предупреждение)
```txt
Сейчас для решения вопроса потребуется перенастроить роутер
Должен предупредить:
- Интернет на некоторое время перестанет работать
- Если возникнут трудности при перенастройке - потребуется выезд техника

Удобно будет сейчас уделить время?
```

#### Инструкция по перенастройке
```txt
1. Нажмите кнопку reset на задней панели и удерживайте ее, пока не начнут мигать индикаторы. Это занимает около 10 секунд
Для предотвращения случайного нажатия эта маленькая кнопка расположена в углублении. Нажмите на нее с помощью стержня ручки, скрепки или зубочистки
2. Отпустите кнопку
3. Дождитесь окончания перезагрузки роутера
4. Подключитесь к вашей wi-fi сети по паролю, который указан на роутере

Если сейчас сбросим настройки - роутер обязательно нужно будет настроить заново, чтобы интернет появился```
