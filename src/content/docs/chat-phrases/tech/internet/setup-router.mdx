### Dom.ru Wave
#### Вход в настройки
```txt
1. Подключитесь к сети роутера по кабелю или Wi-Fi
2. Откройте сайт http://192.168.0.1
3. На странице войдите с данными от роутера (они написаны на его этикетке снизу)
```

#### Настройка соединения
<Tabs>
    <TabItem label="DHCP">
        ```txt
        1. Вверху откройте раздел "WAN";
        2. Укажите параметры:
        - Протокол - "IPoE";
        - Тип - "DHCP";
        - Включить DHCP Option 60 - уберите галочку.

        Для сохранения нажмите "Применить изменения"
        ```
    </TabItem>
    <TabItem label="PPPoE">
        ```txt
        1. Вверху откройте раздел "WAN";
        2. Укажите параметры:
        - Протокол - "PPPoE";
        - Имя пользователя и Пароль - введите данные по договору.

        Для сохранения нажмите "Применить изменения"
        ```
    </TabItem>
    <TabItem label="Static IP">
        ```txt
        1. Вверху откройте раздел "WAN";
        2. Укажите параметры:
        - Протокол - "IPoE";
        - Тип - "Статический";
        - IP-адрес, Маска, Шлюз, DNS - введите данные по договору.

        Для сохранения нажмите "Применить изменения"
        ```
    </TabItem>
</Tabs>

#### Смена каналов
```txt
1. Перейдите в раздел "Wi-FI";
2. Выберите вашу частоту сети (2.4ггц / 5 ггц);
2. В значении "Номер канала" укажите канал ...

Для сохранения нажмите "Применить изменения"
```

#### Настройки Wi-Fi
```txt
1. Перейдите в раздел "Wi-FI";
2. Выберите вашу частоту сети (2.4ггц / 5 ггц);
3. Укажите параметр:
- SSID - желаемое название Wi-Fi.

Для изменения пароля от Wi-Fi:
1. Откройте подпункт "Безопасность";
2. Укажите параметр:
- Предварительный общий ключ - желаемый пароль Wi-Fi.

Для сохранения нажмите "Применить изменения"
```

#### Настройка моста
Для прошивки EC220
```txt
1. Перейдите в раздел "Дополнительные настройки" - "Сеть" - "Интернет";
2. Нажмите "Добавить", в появившемся окне настройки типа соединения выберите "Мост" и нажмите "Сохранить";
3. Далее перейдите в раздел "Группировка интерфейсов" и нажмите "Добавить"

В открывшихся настройках выберите порт LAN для подключения VoIP-шлюза и созданное до этого Bridge-соединение
Также нужно задать имя для группы портов (например VoIP). Затем сохраните настройки
```

#### Настройки Wi-Fi
```txt
1. Откройте раздел "Домашняя сеть"
2. Укажите параметры:
- Имя сегмента - оставьте стандартным;
- Беспроводная сеть Wi-FI (2.4GHz) - "Включено";
- Имя сети (SSID) - желаемое название Wi-Fi;
- Защита сети - "WPA2-PSK";
- Пароль - желаемый пароль Wi-Fi;
- Режим работы - "работает постоянно".

Нажмите кнопку "Сохранить"
```

#### Настройка IPv6
```txt
1. Откройте раздел "Интернет" - "Кабель Ethernet";
2. Найдите на странице пункт "Аутентификация у провайдера (PPPoE / PPTP / L2TP)"
3. Нажмите "Показать дополнительные настройки PPPoE";
3. В появившихся настройках поставьте галочку "Включить IPv6".

Нажмите кнопку "Сохранить"
```

#### Вход в настройки
```txt
1. Подключитесь к сети роутера по кабелю или Wi-Fi
2. Откройте сайт http://192.168.88.1
3. На странице войдите с данными от роутера (Login - admin, Password - пусто)
```

#### Настройка соединения
<Tabs>

        ```txt

        3. Укажите параметры:
        - Enabled - включите галочку;
        - Chain - "srcnat";
        - Out. Interface - "ether1";
        - Action - "masquerade".

        ```

#### Смена каналов
```txt
1. Сверху выберите пункт "WebFIG" - "Interfaces";
2. Посередине выберите ваш интерфейс беспроводной сети;
3. Нажмите на него левой кнопкой мыши для редактирования
4. Укажите параметры:
- Channel Width - "20MHz";
- Frequency - "2417".
- Band - выбираем частоту 5GHz
- Channel Width - "Auto (20/40/80)"
- Frequency - "5201".

Для сохранения параметров нажмите "Ok"
```

### Терминалы Dasan
#### Вход в настройки
```txt
1. Подключитесь к сети терминала по кабелю или Wi-Fi;
2. Откройте сайт http://192.168.55.1:80 (могут быть ссылки: http://192.168.55.1:8080 или http://192.168.55.1:85);
3. В появившемся окне авторизации введите: логин - admin, пароль - vertex25/crimas37/corkun28 или crocus23.
```

#### Настройка PPPoE
```txt
1. Откройте раздел "Advanced Setup" - "WAN Connections";
2. Если есть созданное соединение в режиме "Bridge Mode" - удалите его через кнопку "Del";
3. Далее в "WAN Entry" выберите тип подключения "PPPoE" и нажимаем "ADD";
4. В таблице ниже, на новом созданном подключении, жмем "Edit" и в открывшейся таблице указываем следующие данные:
- Status - Activated;
- Username - Имя пользователя по договору;
- Password - Пароль по договору;
- Connection - "Always On";
- 802.1q - "Tag";
- VLAN ID - VLAN по договору;
- Get IP Address - "Dynamic".

Для сохранения параметров нажмите "Apply"
```

#### Настройка Wi-Fi
<Tabs>
    <TabItem label="H660RW">
        ```txt
        1. Откройте раздел "Wi-Fi Setup" - "Wi-Fi Setting";
        2. В строке "Channel" выбираем все каналы (номера);
        3. В строке "Authentication Type" меняем значение на "WPA(WPA2)PSK" и параметр "Encryption" на "AES".

        При желании название сети можно изменить в строчке "SSID", пароль в "Pre-Shared Key"
        Для сохранения параметров нажмите "Apply"
        ```
    </TabItem>
    <TabItem label="H640RW">
        ```txt
        1. Перейдите в раздел "Basic Setting" - "SSID Setting";
        2. Для изменения параметров нажмите "Modify";
        3. В строке "Authentication" поменяйте значение на "WPA2-PSK/WPA2-Personal" и параметр "Encryption" на "AES".

        При желании название сети можно изменить в строчке "SSID", пароль в "Pre-Shared Key"
        Для сохранения параметров нажмите "Apply", затем вверху кнопку "Apply Config"
        ```
    </TabItem>
</Tabs>

#### Смена каналов
```txt
1. Откройте раздел "Wi-Fi Setup" - "Wi-Fi Setting";
2. В строке "Channel" поменяйте значение на "Manual" и выберите номер канала.

Для сохранения параметров нажмите "Apply"
```

#### Проброс портов
<Tabs>
    <TabItem label="H660RW">
        ```txt
        1. Откройте раздел "Firewall Setup" - "NAT" - "Port Forwarding"

        ```
    </TabItem>

        ```txt

        - "Target IP" - IP-адрес устройства, для которого открываем порт;
        - "Protocol" - "TCP/UDP"
        ```
