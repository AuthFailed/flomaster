  <starlight.TabItem label="iOS" icon="apple">
    ```txt
    1. Откройте настройки Wi-Fi на телефоне
    2. В самом верху найдите сеть с галочкой слева. Нажмите на значок "i" справа от нее
    3. Пролистайте страницу вниз до пунктов IP-адрес, маска подсети и маршрутизатор

    На открывшейся странице найдите значения IP-адрес, маска подсети, маршрутизатор и пришлите фото/скриншот данных в чат
    ```
  </starlight.TabItem>


  </starlight.TabItem>

  <starlight.TabItem label="Mac" icon="apple">
    ```txt

    2. Нажмите на ​Замок ​для разблокировки параметров
    3. Введите пароль пользователя и нажмите ​"Снять защиту"
    4. Откройте раздел ​"Дополнительно" - ​"TCP/IP" - "Использовать DHCP"

    На открывшейся странице найдите IP-адреса и пришлите фото/скриншот данных в чат
    ```
  </starlight.TabItem>

### Установка статических метрик

<starlight.Tabs>
  <starlight.TabItem label="Android" icon="linux">
    ```txt
    1. Откройте настройки Wi-Fi на телефоне
    2. Нажмите на шестеренку рядом с вашей сетью
    3. Вверху справа нажмите на карандаш - "Расширенные настройки" - "Настройки IP-протокола"
    4. Укажите параметры:
    - Статический IP-адрес
    - IP-адрес - [ПИШИ]
    - Шлюз - [ПИШИ]
    - DNS 1 - [ПИШИ]
    - DNS 2 - [ПИШИ]

    Нажмите кнопку "Сохранить" 
    ```
  </starlight.TabItem>

  <starlight.TabItem label="iOS" icon="apple">
    ```txt
    1. Откройте настройки Wi-Fi на телефоне
    2. В самом верху найдите сеть с галочкой слева. Нажмите на значок "i" справа от нее
    3. Откройте раздел "Настройка IP" - "Вручную"
    4. Укажите параметры:
    - IP-адрес - [ПИШИ]
    - Маска подсети - [ПИШИ]
    - Маршрутизатор - [ПИШИ]

    Нажмите кнопку "Сохранить"
    ```
  </starlight.TabItem>

  <starlight.TabItem label="Windows" icon="seti:windows">
    ```txt
    1. Нажмите на клавиатуре WIN+R, в окне введите "ncpa.cpl" без кавычек
    2. В открывшемся окне нажмите правой кнопкой мыши на "Подключение по локальной сети" или "Ethernet" - "Свойства" - "IP версии 4 (TCP/IP)" - "Свойства"
    3. Укажите параметры:
    - IP-адрес - [ПИШИ]
    - Маска подсети - [ПИШИ]
    - Основной шлюз - [ПИШИ]
    - Предпочитаемый DNS - [ПИШИ]
    - Альтернативный DNS - [ПИШИ]

    Нажмите кнопку "ОК"
    ```
  </starlight.TabItem>

  <starlight.TabItem label="Mac" icon="apple">
    <starlight.TabItem label="Wi-Fi">
      ```txt
      1. Откройте "Системные настройки" - "Wi-FI"
      2. Нажмите "Подробнее" рядом с сетью, к которой выполнено подключение
      3. Слева откройте пункт "TCP/IP"
      4. Укажите параметры:
      - Конфигурация IP - "Вручную"
      - IP-адрес - [ПИШИ]
      - Маска подсети - [ПИШИ]
      - Маршрутизатор - [ПИШИ]
      5. Перейдите в пункт "DNS", нажмите кнопку "+" в строке добавления DNS
      6. Укажите параметры:
      - Первый DNS - [ПИШИ]
      - Второй DNS - [ПИШИ]

      Параметры сохраняем нажатием кнопки "Ок"
      ```
    </starlight.TabItem>

    <starlight.TabItem label="Патчкорд">
      ```txt
      1. Откройте "Системные настройки" - "Сеть"
      2. Нажмите на три точки справа - "Добавить службу" 
      3. Имя службы введите "dom.ru", нажмите "Создать"
      4. В списке сетевых служб выберите новую созданную Вами службу, затем нажмите кнопку "Подробнее"
      5. Откройте пункт "TCP/IP"
      6. Укажите параметры:
      - Конфигурация IP - "Вручную"
      - IP-адрес - [ПИШИ]
      - Маска подсети - [ПИШИ]
      - Маршрутизатор - [ПИШИ]
      7. Перейдите в пункт "DNS", нажмите кнопку "+" в строке добавления DNS
      8. Укажите параметры:
      - Первый DNS - [ПИШИ]
      - Второй DNS - [ПИШИ]

      Параметры сохраняем нажатием кнопки "Ок"
      ```
    </starlight.TabItem>
  </starlight.TabItem>
</starlight.Tabs>
