### Настройка PPPoE
<Tabs>
    <TabItem label="Win7">
        ```txt
        1. Открываем Пуск, затем Панель управления;
        2. На панели управления открываем раздел Сеть и Интернет;
        3. Выбираем Центр управления сетями и общим доступом;
        4. Выбираем Настройка нового подключения или сети;
        5. В появившемся окне выбираем пункт Подключение к Интернету;
        6. Появится выбор высокоскоростного подключения. Нажимаем на Высокоскоростное (с PPPoE);
        7. Вводим имя пользователя и пароль от PPPoE, нажимаем Подключить.

        После этого PPPoE соединение будет создано
        ```
    </TabItem>

### Настройка Static IP
<Tabs>
    <TabItem label="Win">
        ```txt
        1. Нажимаем на клавиатуре WIN+R;
        2. В окне вводим "ncpa.cpl" без кавычек;
        3. В открывшемся окне нажимаем правой кнопкой мыши на "Подключение по локальной сети" или "Ethernet", жмем
        "Свойства";
        4. В списке ищем "Протокол Интернета версии 4" (или "IP версии 4"), дважды нажимаем на него;
        5. В окне выбираем "Использовать следующий IP адрес" и вводим данные из смс.

        После этого Static IP соединение будет создано
        ```
    </TabItem>
    <TabItem label="Mac">
        ```txt
        1. Переходим в "Системные настройки", раздел "Сеть";
        1) Открываем всплывающее меню действий справа (в виде трех точек), затем выбираем "Добавить службу";
        2) Оставляем интерфейс Ethernet, имя службы вводим на свое усмотрение (Например, dom.ru). Жмем "Создать";
        3) В списке сетевых служб выбираем новую созданную вами службу, затем нажимаем кнопку "Подробнее";
        4) Открываем пункт TCP/IP, конфигурацию IP выставляем в режим "Вручную" и вводим данные из СМС;
        5) Переходим в пункт "DNS", жмем на кнопку "+" в строке добавления DNS, данные также указываем из СМС.

        После этого Static IP соединение будет создано
        ```
    </TabItem>
</Tabs>
