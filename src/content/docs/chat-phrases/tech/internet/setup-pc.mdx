### Настройка PPPoE
<Tabs>
    <TabItem label="Win7">
        ```txt
        Для настройки соединения:
        1. Откройте "Пуск" - "Панель управления" - "Сеть и Интернет";
        2. Выберите "Центр управления сетями и общим доступом" - "Настройка нового подключения или сети";
        3. В появившемся окне выберите пункт "Подключение к Интернету";
        4. Нажмите на "Высокоскоростное (с PPPoE)" и введите имя пользователя и пароль по договору
        5. Нажмите "Подключить".

        После этого PPPoE соединение будет создано
        ```
    </TabItem>

    </TabItem>

        ```txt
        3. Оставьте интерфейс Ethernet, имя службы введите на свое усмотрение (Например, dom.ru), далее "Создать";

        6. Включите параметр "Запомнить пароль", затем введите пароль.

        После этого PPPoE соединение будет создано
        ```
    </TabItem>

### Настройка Static IP
<Tabs>
    <TabItem label="Win">
        ```txt
        Для настройки соединения:
        1. Нажмите на клавиатуре WIN+R, в окне введите "ncpa.cpl" без кавычек;
        2. В открывшемся окне нажмите правой кнопкой мыши на "Подключение по локальной сети" или "Ethernet" - "Свойства";
        3. В списки найдите "Протокол Интернета версии 4" (или "IP версии 4"), дважды нажмите на него;
        4. В окне выберите "Использовать следующий IP адрес" и введите данные по договору.

        После этого Static IP соединение будет создано
        ```
    </TabItem>
    <TabItem label="Mac">
        ```txt
        Для настройки соединения:
        1. Откройте раздел "Системные настройки" - "Сеть";
        2. Откройте меню действий справа (в виде трех точек), выберите "Добавить службу";
        3. Оставьте интерфейс Ethernet, имя службы введите на свое усмотрение (Например, dom.ru), далее "Создать";
        4. В списке сетевых служб выберите созданную вами службу, нажмите кнопку "Подробнее";
        5. Откройте пункт "TCP/IP", "Конфигурация IP" установите в "Вручную" и введите данные по договору;
        6. Перейдите в пункт "DNS", нажмите на кнопку "+" в строке добавления DNS, данные также введите по договору.

        После этого Static IP соединение будет создано
        ```
    </TabItem>
</Tabs>
