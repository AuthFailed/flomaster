### Подключение CAM-модуля

```txt
Чип на карте можно протереть чистой сухой тканью, затем подключите всё обратно:
* SMART-карта вставляется в CAM-модуль чипом вперед
* Логотипы на карте и на модуле направлены в одну сторону
* CAM-модуль вставляется логотипом против лицевой стороны дисплея
```

### Автопоиск каналов

```txt
Потребуется перенастройка каналов. Пройдите по инструкции:
1. Переподключите антенный кабель к ТВ в соответствующий порт и проведите перенастройку каналов
2. При перенастройке укажите параметры:
- Тип поиска - "Полный"
- Источник сигнала - "Кабель (DVB-C)"
- Тип каналов - "Цифровые и аналоговые"
- Страна (если запросит) - "Финляндия" или "Германия"
```

### Ручной поиск по частотам

```txt
Требуется перенастройка каналов по частотам. Пройдите по инструкции:
1. Переподключите антенный кабель к ТВ в соответствующий порт и проведите перенастройку каналов
2. При перенастройке укажите параметры:
- Тип поиска - "Полный"
- Источник сигнала - "Кабель (DVB-C)"
- Тип каналов - "Цифровые и аналоговые"
- Страна (если запросит) - "Финляндия" или "Германия"
- Поставщик услуг / Провайдер (если запросит) - "Другое" или "Прочее"
- Начальная частота - 250 МГц (250 000 КГц) 
- Конечная частота - 450 Мгц (450 000 КГц)
- Модуляция - 256 QAM
- Символьная скорость - 6875
```
