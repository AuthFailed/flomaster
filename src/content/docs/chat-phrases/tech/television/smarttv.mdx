### Перезапуск приложения Movix
```txt
Перейдите в основное меню приложения, где отображаются баннеры с каналами\подписками, затем зайдите в настройки (значок шестеренки в правом верхнем углу).

Там будет раздел "Настройки системы" и в нем выберите пункт "Перезапустить приложение"
```

### Проверка привязанных устройств
```txt
Перейдите в основное меню приложения, где отображаются баннеры с каналами\подписками, затем зайдите в настройки (значок шестеренки в правом верхнем углу)

Там будет раздел "Устройства". Справа будут отображены привязанные устройства, где должен быть Ваш ТВ
```

### Проверка тех. информации (Модель ТВ/Версия приложения)
```txt
Перейдите в основное меню приложения, где отображаются баннеры с каналами\подписками, затем зайдите в настройки (значок шестеренки в правом верхнем углу)

Там будет раздел "Настройки системы" и в нем подраздел "Техническая информация"
Нажмите на последний пункт и сделайте фото информации
```

### Смена разрешения
```txt
Нажмите во время вещания кнопку вниз или ОК. Откроется навигационное меню. Выберите справа значок шестеренки и нажмите на нем кнопку ОК

Откроется меню, выберите в верхней строке разрешение (720p или 576p), которое стоит перед последним пунктом "Max", потом нажмите на нем ОК и ниже нажмите кнопку "Применить"
```

### Проверка скорости через Movix
```txt
Перейдите в основное меню приложения, где отображаются баннеры с каналами\подписками, затем зайдите в настройки (значок шестеренки в правом верхнем углу).

Там будет раздел "Настройки системы". В него зайдите, будет строка "Замер скорости". Проведите замер и напишите результаты
```
