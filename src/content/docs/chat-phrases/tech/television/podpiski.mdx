# IVI

#### Авторизация

```txt
1. В главном меню приставки найдите раздел "Все подписки" и откройте его
2. Найдите приложение "IVI" и нажмите на него
3. Выберите любой фильм/сериал и откройте
4. Нажмите кнопку "Смотреть в IVI"

Появится окно установки приложения - согласитесь с установкой
После установки выполняем авторизуйтесь по номеру телефона
```

#### Фикс ошибки при авторизации

При вечной авторизации после ввода кода из СМС

```
1. Откройте приложение "IVI" на приставке, должно появиться окно для ввода кода из СМС
2. Данные не вводите, нажмите кнопку "Назад"
3. Справа вверху нажмите "Войти", затем "Войдите или зарегистрируйтесь"

Выполните авторизацию по номеру телефона и кода из СМС
```


#### Проверка обновлений на приставках

```txt
1. Зайдите в раздел "Приложения" в правом верхнем углу
2. Откройте "Play Маркет" - войдите в свой профиль или создайте его
3. Перейдите в пункт "Управление приложениями и играми" - "Проверить обновления" - "IVI"

Уточните, есть ли доступные обновления?
```

### Кинопоиск

#### Активация подписки

В случаях, когда номер указан корректно, но в Яндексе отображается, что подписка неактивна

```txt
1. Откройте страницу plus.yandex.ru и авторизуйтесь по номеру телефона [ПИШИ]
2. После откройте страницу plus.yandex.ru/domru

На открывшеся странице выберите аккаунт, на который хотите активировать подписку
```

#### Проверка обновлений на приставках

```
1. Зайдите в раздел "Приложения" в правом верхнем углу
2. Откройте "Play Маркет" - войдите в свой профиль или создайте его
3. Перейдите в пункт "Управление приложениями и играми" - "Проверить обновления" - "Кинопоиск"

Уточните, есть ли доступные обновления?
```
