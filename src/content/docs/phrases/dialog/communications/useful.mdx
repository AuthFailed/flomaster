### Загрузка фото через сайт

<starlight.Tabs>
  <starlight.TabItem label="ImgBB">
    ```txt
    Загрузите картинку на сайт https://ru.imgbb.com
    Как загрузится, отправьте сюда ссылку, которая появится по центру страницы
    ```
  </starlight.TabItem>

  <starlight.TabItem label="PastePics">
    ```txt
    Загрузите картинку на сайт https://ru.paste.pics
    Как загрузится, отправьте сюда ссылку, которая появится сверху страницы
    ```
  </starlight.TabItem>

  <starlight.TabItem label="Postimages">
    ```txt
    Загрузите картинку на сайт https://postimages.org/ru
    Как загрузится, отправьте сюда ссылку, которая будет находиться в поле "Ссылка" или "Прямая ссылка"
    ```
  </starlight.TabItem>
</starlight.Tabs>
