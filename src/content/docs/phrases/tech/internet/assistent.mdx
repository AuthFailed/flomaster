### Предложение удаленного подключения
```txt
Мы можем подключиться к вашему компьютеру для выполнения удаленной диагностики через приложение "Ассистент"
Удобно будет так сделать?
```

### Установка Ассистента
```txt
Для установки Ассистента:
1. Перейдите на сайт https://мойассистент.рф/скачать
2. На открывшейся странице нажмите "Скачать Ассистент "Быстрый старт""

После загрузки запустите приложение и напишите в чат "Идентификатор" и "Пароль"
```

### Запрос данных для подключения
```txt
Сейчас нужно уточнить данные для подключения:
В программе указаны "Идентификатор" и "Пароль"
Найдите их и напишите значения в чат
```
