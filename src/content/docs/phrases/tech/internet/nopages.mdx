### Проверка получаемых метрик

<starlight.Tabs>
  <starlight.TabItem label="Android" icon="linux">
    ```txt
    1. Откройте настройки Wi-Fi на телефоне
    2. Нажмите на шестеренку рядом с вашей сетью

    На открывшейся странице найдите значения IP-адрес, Шлюз, Маска подсети и пришлите скриншот данных в чат 
    ```
  </starlight.TabItem>

  <starlight.TabItem label="iOS" icon="apple">
    ```txt
    1. Откройте настройки Wi-Fi на телефоне
    2. В самом верху найдите сеть с галочкой слева. Нажмите на значок "i" справа от нее
    3. Пролистайте страницу вниз до пунктов IP-адрес, маска подсети и маршрутизатор

    На открывшейся странице найдите значения IP-адрес, маска подсети, маршрутизатор и пришлите фото/скриншот данных в чат
    ```
  </starlight.TabItem>

  <starlight.TabItem label="Windows" icon="seti:windows">
    ```txt
    1. Нажмите на клавиатуре WIN+R, в окне введите "ncpa.cpl" без кавычек
    2. Нажмите правой кнопкой мыши на "Беспроводном сетевом соединении" - "Состояние"

    На открывшейся странице найдите значения IP-адрес, маска подсети и пришлите фото/скриншот данных в чат
    ```
  </starlight.TabItem>

  <starlight.TabItem label="Mac" icon="apple">
    ```txt
    1. Нажмите на значок подключения​ в панели задач и выберите строку "Открыть настройки сети"​
    2. Нажмите на ​Замок ​для разблокировки параметров
    3. Введите пароль пользователя и нажмите ​"Снять защиту"
    4. Откройте раздел ​"Дополнительно" - ​"TCP/IP" - "Использовать DHCP"

    На открывшейся странице найдите IP-адреса и пришлите фото/скриншот данных в чат
    ```
  </starlight.TabItem>
</starlight.Tabs>

### Установка статических метрик

<starlight.Tabs>
  <starlight.TabItem label="Android" icon="linux">
    ```txt "[IP КЛИЕНТА]" "[ШЛЮЗ КЛИЕНТА]" "[DNS1 КЛИЕНТА]" "[DNS2 КЛИЕНТА]"
    1. Откройте настройки Wi-Fi на телефоне
    2. Нажмите на шестеренку рядом с вашей сетью
    3. Вверху справа нажмите на карандаш - "Расширенные настройки" - "Настройки IP-протокола"
    4. Укажите параметры:
    - Статический IP-адрес
    - IP-адрес - [IP КЛИЕНТА]
    - Шлюз - [ШЛЮЗ КЛИЕНТА]
    - DNS 1 - [DNS1 КЛИЕНТА]
    - DNS 2 - [DNS2 КЛИЕНТА]

    Нажмите кнопку "Сохранить" 
    ```
  </starlight.TabItem>

  <starlight.TabItem label="iOS" icon="apple">
    ```txt "[IP УСТРОЙСТВА]" "[МАСКА ПОДСЕТИ]" "[IP РОУТЕРА]"
    1. Откройте настройки Wi-Fi на телефоне
    2. В самом верху найдите сеть с галочкой слева. Нажмите на значок "i" справа от нее
    3. Откройте раздел "Настройка IP" - "Вручную"
    4. Укажите параметры:
    - IP-адрес - [IP УСТРОЙСТВА]
    - Маска подсети - [МАСКА ПОДСЕТИ]
    - Маршрутизатор - [IP РОУТЕРА]

    Нажмите кнопку "Сохранить"
    ```
  </starlight.TabItem>

  <starlight.TabItem label="Windows" icon="seti:windows">
    ```txt "[IP АДРЕС]" "[МАСКА ПОДСЕТИ]" "[ШЛЮЗ КЛИЕНТА]" "[DNS1 КЛИЕНТА]" "[DNS2 КЛИЕНТА]"
    1. Нажмите на клавиатуре WIN+R, в окне введите "ncpa.cpl" без кавычек
    2. В открывшемся окне нажмите правой кнопкой мыши на "Подключение по локальной сети" или "Ethernet" - "Свойства" - "IP версии 4 (TCP/IP)" - "Свойства"
    3. Укажите параметры:
    - IP-адрес - [IP АДРЕС]
    - Маска подсети - [МАСКА ПОДСЕТИ]
    - Основной шлюз - [ШЛЮЗ КЛИЕНТА]
    - Предпочитаемый DNS - [DNS1 КЛИЕНТА]
    - Альтернативный DNS - [DNS2 КЛИЕНТА]

    Нажмите кнопку "ОК"
    ```
  </starlight.TabItem>

  <starlight.TabItem label="Mac" icon="apple">

### Удаление сети Wi-Fi

<starlight.Tabs>
  <starlight.TabItem label="Android" icon="linux">
    ```txt
    1. Откройте настройки Wi-Fi на телефоне
    2. Нажмите на шестеренку рядом с вашей сетью
    3. Нажмите кнопку "Удалить"
    ```
  </starlight.TabItem>

  <starlight.TabItem label="iOS" icon="apple">
    ```txt
    1. Откройте настройки Wi-Fi на телефоне
    2. В самом верху найдите сеть с галочкой слева. Нажмите на значок "i" справа от нее
    3. В открывшемся окне нажмите "Забыть эту сеть" и подтвердите
    ```
  </starlight.TabItem>

  <starlight.TabItem label="Windows" icon="seti:windows">
    <starlight.Tabs>
      <starlight.TabItem label="Win10">
        ```txt
        1. Нажмите на иконку подключения к интернету на панели уведомлений (в правом нижнем углу)
        2. Откройте "Параметры сети" - "Управление параметрами сети Wi-Fi"
        3. Выберите из списка Wi-Fi вашу сеть и нажмите кнопку "Забыть"
        ```
      </starlight.TabItem>

      <starlight.TabItem label="Win8">
        ```txt
        1. Откройте список доступных Wi-Fi сетей
        2. Нажмите на вашу сеть правой кнопкой мыши и нажмите "Забыть эту сеть"
        ```
      </starlight.TabItem>

      <starlight.TabItem label="Win7">
        ```
        1. Откройте "Пуск" - "Панель управления" - "Сеть и интернет" - "Центр управления сетями и общим доступом"
        2. Слева откройте "Управление беспроводными сетями"
        3. Найдите вашу сеть в списке и удалите ее

        ```
      </starlight.TabItem>
    </starlight.Tabs>
  </starlight.TabItem>

  <starlight.TabItem label="Apple" icon="apple">
    ```txt
    1. Откройте "Системные настройки" - "Wi-Fi"
    2. Нажмите "Подробнее" рядом с сетью, которую вы используете
    3. Нажмите "Забыть эту сеть", а затем "Удалить" для подтверждения
    ```
  </starlight.TabItem>
</starlight.Tabs>
