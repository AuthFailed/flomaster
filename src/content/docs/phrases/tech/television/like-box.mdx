### LikeBox

#### Проверка CA ID

<starlight.Tabs>
    <starlight.TabItem label="Через настройки">
        ```txt
        Нужно уточнить номер приставки:
        1. Нажмите на пульте "Меню", далее зайдите в настройки (иконка шестеренки)
        2. Откройте раздел "Система" - "Техинфо"

        На открывшейся странице найдите значение "CA ID"
        Напишите его, пожалуйста, или пришлите его фото в чат
        ```
    </starlight.TabItem>

    <starlight.TabItem label="С наклейки">
        ```txt
        Нужно уточнить номер приставки:
        Снизу на приставке, где наклейка, будет написан её CA ID формата XXXXXXXXXXXXX
        Напишите его, пожалуйста, или пришлите фото наклейки в чат
        ```
    </starlight.TabItem>
</starlight.Tabs>

#### Перенастройка приставки

```txt
Потребуется перенастройка приставки:
1. Нажмите на пульте кнопку "Меню", далее зайдите в настройки (иконка шестеренки)
2. Откройте раздел "Настройки системы" - "Сброс системы до заводских настроек"
3. Подтвердите сброс

После сброса пройдите по инструкции на экране для настройки приставки
```

#### Проверка и обновление ПО

```txt
Потребуется проверить наличие обновлений ПО:
1. Нажмите на пульте кнопку "Меню", далее зайдите в настройки (иконка шестеренки)
2. Откройте раздел "Настройки системы" - "Обновления"

Если обновление не требуется - появится соответствующее сообщение и версия установленного ПО
```

#### Проверка разрешения

```txt
Проверим установленное разрешение:
1. Нажмите во время эфира на кнопку "ОК", либо "Вниз"
2. Нажмите на шестеренку, далее "Настройки параметров эфира"
3. В верхней строке измените разрешение эфира на 720p или 576p

Нажмите кнопку "ОК" и "Применить"
```

#### Проверка MAC-адреса

<starlight.Tabs>
    <starlight.TabItem label="Через настройки">
        ```txt
        Нужно уточнить номер приставки:
        Нажмите на пульте от приставке 911 во время эфира и найдите в окне MAC-адрес формата XX:XX:XX:XX:XX:XX
        Напишите его, пожалуйста, или пришлите фото наклейки в чат
        ```
    </starlight.TabItem>

    <starlight.TabItem label="С наклейки">
        ```txt
        Нужно уточнить номер приставки:
        Снизу на приставке, где наклейка, будет написан её MAC-адрес формата XX:XX:XX:XX:XX:XX
        Напишите его, пожалуйста, или пришлите фото наклейки в чат
        ```
    </starlight.TabItem>

#### Перенастройка приставки

```txt
Потребуется перенастройка приставки:
1. Нажмите иконку шестеренки в главном меню приставки
2. Откройте раздел "Настройки системы" - "Хранилище и сброс" - "Заводские настройки" - "Удалить все"
3. Дождитесь перезагрузки приставки и выполните первоначальную настройку:
- Выберите "Первое включение" и "Только сейчас"
- Выберите язык интерфейса
- Настройте разрешение экрана
- Отрегулируйте масштаб изображения кнопками Вверх/Вниз
- Выберите способ подключения (Ethernet или Wi-Fi)
- Установите дату и часовой пояс
```

#### Проверка и обновление ПО

```txt
Потребуется проверить наличие обновлений ПО:
1. Нажмите иконку шестеренки в главном меню приставки
2. Откройте раздел "Настройки системы" - "Об устройстве" - "Обновление системы" - "Проверить наличие обновлений"
3. Дождитесь результата проверки:
- При наличии обновлений начнётся загрузка и установка нового ПО
- При отсутствии обновлений появится соответствующее уведомление
```

#### Проверка разрешения

```txt
Проверим установленное разрешение:
1. Нажмите во время эфира на кнопку "ОК", либо "Вниз"
2. Нажмите на значок шестеренки, далее "Настройки параметров эфира"
3. В верхней строке измените разрешение эфира на 720p или 576p

Нажмите кнопку "ОК" и "Применить"
```

#### Отключение HDMI-CEC

```txt
Потребуется отключить HDMI-CEC:
1. Нажмите иконку шестеренки в главном меню приставки
2. Откройте раздел "Настройки" - "Настройки системы" - "HDMI-CEC"

В открывшемся окне найдите пункт "Включить HDMI-CEC" и выключите его
```

#### Очистка кеша

```txt
Сейчас для очистки кеша выполните действия:
1. Нажмите иконку шестеренки в главном меню приставки
2. Откройте раздел "Настройки" - "Настройки системы" - "Приложения" - "Movix Pro"
3. Последовательно нажмите:
- "Удалить данные"
- "Удалить кеш"
4. Перезагрузите приставку:
- Отключите питание на 30 секунд
- Подключите питание обратно
- Проверьте работу приложения
```

#### Проверка сетевых метрик

```txt
Проверим статус сети на приставке:
1. Нажмите иконку шестеренки в главном меню приставки
2. Откройте раздел "Настройки" - "Настройки системы" - "Сеть"
3. В разделе "Сеть" найдите строку "Подключено" и нажмите на нее

Напишите в чат указанные IP-адреса
```

#### Включение родительского контроля

```txt
Для включения родительского контроля:
1. Нажмите иконку шестеренки в главном меню приставки
2. Откройте раздел "Настройки" - "Родительский контроль" - "Подтверждение покупки Пин-кодом"
3. Активируйте функцию подтверждения покупки
```
