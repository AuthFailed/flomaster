### Переустановка Movix

<starlight.Tabs>
    <starlight.TabItem label="Samsung">
        ```txt
        Сейчас нужно переустановить приложение Movix, пройдите по инструкции:
        1. Нажмите кнопку Home (Домой) на пульте
        2. Снизу выберите раздел Apps (Приложения)
        3. В открывшемся магазине приложений справа сверху выберите раздел настроек
        4. В перечне установленных приложений выберите "Дом.ru Movix" - "Удалить"

        Далее перезагрузите телевизор, зайдите в магазин приложений, нажмите справа сверху лупу, найдите "Дом.ru Movix" и установите

        ```

### Перезапуск приложения Movix

```txt
Давайте перезапустим приложение:
1. Нажмите иконку шестеренки в главном меню приложения
2. Откройте раздел "Настройки системы" - "Перезапустить приложение"
и подтвердите действие
```

### Проверка привязанных устройств

```txt
Проверим привязку устройства:
1. Нажмите иконку шестеренки в главном меню приложения
2. Откройте раздел "Устройства"

На странице будут отображены привязанные устройства, где должен быть Ваш ТВ
Если его там нет - нажмите на "+ Добавить это устройство"
```

### Проверка тех. информации (Модель ТВ/Версия приложения)

```txt
Сейчас нужно проверить модель ТВ и версию приложения:
1. Нажмите иконку шестеренки в главном меню приложения
2. Откройте раздел "Настройки системы" - "Техническая информация"

На открывшейся странице нажмите на последний пункт и сделайте фото информации
```

### Смена разрешения

```txt
Для смены разрешения:
1. Во время вещания канала нажмите кнопку вниз или "ОК"
2. В открывшемся меню нажмите на значок шестеренки

В списке разрешенией выберите разрешение, которое стоит перед пунктом "Max", далее нажмите кнопку "Применить"
```

### Проверка скорости через Movix

```txt
Проверим скорость через приложение:
1. Нажмите иконку шестеренки в главном меню приложения
2. Откройте раздел "Настройки системы" - "Замер скорости"

Проведите замер на открывшейся странице и напишите результат
```
