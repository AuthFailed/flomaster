### Проверка версии МП
```txt
В приложении "Умный Дом.ру" откройте раздел "Кабинет", далее пункт "О приложении"
Уточните, какая версия приложения там указана?
```

### Добавление камеры в МП
```txt
Для добавления камеры:
1. Находясь на главном экране, в блоке "Мой дом", нажмите на иконку "+"
2. Выберите из списка модель вашей камеры;
3. Появится окно выбора тарифа, выбираем указанный по договору тариф и принимаем условия Публичной оферты;
4. Выберите способ подключения камеры к сети, кабелем или Wi-Fi и подключитесь к сети.
```

### Пинг до vn.domru.ru
<starlight.Tabs>
    <starlight.TabItem label="Windows" icon="seti:windows">
        ```txt
        Нажмите комбинацию клавиш WIN+R, в открывшемся окне напишите "cmd" без кавычек и нажмите Enter
        Откроется черное окно и строка ввода команды

        Введите команду "ping vn.domru.ru -n 30 -l 1200" без кавычек и нажмите клавишу Enter
        После "ping" идет пробел. Перед "-n" также пробел, и перед 30 пробел. Перед 1200 буква L прописная

        По окончании выполнения команды внизу должен отобразиться процент потерь и количество полученных и потерянных
        пакетов
        Пришлите результат текстом или фотографией результатов
        ```
    </starlight.TabItem>
