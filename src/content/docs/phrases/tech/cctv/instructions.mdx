### Подключение камеры в МП
```txt
Для подключения камеры:
1. Находясь на главном экране, в блоке "Мой дом", нажмите на иконку "+" в верхнем углу
2. Выберите из списка модель вашей камеры
3. Появится окно выбора тарифа и типа записи, выбираем указанный по договору тариф и принимаем условия Публичной оферты
4. Выберите способ подключения камеры к сети, кабелем или Wi-Fi и подключитесь к сети
```

### Управление настройками камер
```txt
Для управления настройками камеры необходимо нажать на иконку настроек в горизонтальном меню на экране камеры (меню меняет расположение в зависимости от ориентации экрана)
```

### Настройка уведомлений камеры
```txt
Для управления настройками уведомлений:
1. Нажмите кнопку "Оповещения" в настройках камеры
2. На экране "Оповещения" можете настроить уведомления для пуш-уведомлений и журнала событий
```

### Просмотр видео онлайн
```txt
Для просмотра видео с камеры нажмите на превью камеры в разделе "Мой дом" на главном экране

В открывшемся окне будет экран плеера камеры в режиме онлайн
```

### Просмотр архива видео камеры
```txt
Для просмотра архива видео с камер:
1. В разделе “Мой дом” откройте интересующую вас камеру
2. Откроется экран архива с возможностью выбора даты и события

— Просмотр архива выполняется перемещением ползунка времени в нужную сторону
— Возврат к прямой трансляции осуществляется нажатием на кнопку live
— Для быстрого перемещения на нужную дату нажмите на кнопку с датой
— Отдельные события можно просмотреть, нажав на кнопку проигрывания напротив события в ленте событий
```

### Скачивание фрагмента архива видео
```txt
Для скачивания архива:
1. Нажмите на кнопку "Архив" на экране плеера
2. Откроется экран архива с возможностью выбора начала и конца отрезка видео и кнопкой "Скачать”
3. После нажатия на кнопку видеофрагмент будет загружен и станет доступен в галерее устройства
```

### Удаление видео с архива
```txt
Архив записей удаляется только по истечению срока хранения, до окончания этого срока фрагмент удалить не получится. Срок хранения архива зависит от вашего тарифа
```

### Настройка чувствительности камеры при записи по движению
```txt
Для настройки чувствительности камеры:
1. Перейдите в настройки камеры
2. В блоке "Настройки устройства" выберите "Датчик движения"

На открывшемся экране в поле "Чувствительность камеры" указано текущее значение чувствительности
Можете регулировать чувствительность камеры передвигая ползунок и нажимая на кнопку "Применить"
```

### Передача звука с приложения на камеру (РТТ)
```txt
Для управления двусторонней связью:
1. Нажмите на иконку микрофона в нижней панели
2. Удерживайте кнопку на экране смартфона в процессе разговора

Если все сделали верно, на экране появится шкала громкости звука
```

### Настройка громкости микрофона
```txt
Для настройки громкости микрофона на камере:
1. Нажмите на иконку в режиме просмотра видео в плеере
2. Отрегулируйте громкость кнопками (справа - громче, слева - тише)
```

### Поворот изображения с камеры
```txt
Для управления поворотом камеры:
1. Нажмите на кнопку "Положение камеры"
2. Используйте кнопки для  поворота изображения:
— "Повернуть изображение" - изображение повернется на 180 градусов по вертикали
— "Отобразить зеркально" - изображение повернется на 180 градусов по горизонтали
```

### Управление поворотным механизмом камеры
```txt
Для управления поворотным механизмом на камере (если камера оснащена данной возможностью):
1. Во время просмотра видео в плеере внизу нажмите на иконку со стрелками
2. Откроется джойстик для управления камерой

При нажатии на стороны джойстика камера изменит положение
```
